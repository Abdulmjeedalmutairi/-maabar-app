# تحديث لوحة التاجر - Dashboard

## التغييرات التي تمت:

### 1. المكونات الجديدة (`src/components/dashboard/`):
- **StatCard.js** - بطاقة إحصائية قابلة للنقر (طلبات، رسائل، عروض، استفسارات)
- **QuickAction.js** - إجراء سريع للوظائف الشائعة
- **StatusBar.js** - شريط تقدم حالة الطلب (8 مراحل)
- **RequestCard.js** - بطاقة عرض الطلب مع التفاصيل
- **DashboardOverview.js** - تبويب "نظرة عامة" كامل

### 2. الشاشة الجديدة (`src/screens/buyer/DashboardScreen.js`):
- لوحة تاجر كاملة مطابقة لموقع maabar.io
- دعم ثلاثي اللغات (عربي، إنجليزي، صيني)
- Real-time updates عبر Supabase
- تصميم متجاوب للجوال

### 3. التحديثات:
- **BuyerTabs.js** - تحديث لاستخدام DashboardScreen بدل HomeScreen
- الألوان (`src/lib/colors.js`) - نفس ألوان الموقع

## ميزات لوحة التاجر الجديدة:

### تبويب "نظرة عامة" (Overview):
- **4 بطاقات إحصائية**: طلبات، رسائل، عروض، استفسارات
- **إجراءات تحتاج انتباه**: عروض تحتاج رد، طلبات تحتاج دفع
- **4 إجراءات سريعة**: رفع طلب قياسي، تصفح المنتجات، طلباتي، Private Label
- **طلباتي الأخيرة**: آخر 3 طلبات مع حالة كل طلب

### التكاملات:
- **Supabase Real-time**: تحديث تلقائي عند تغيير العروض/الرسائل
- **التنقل**: تكامل كامل مع navigation الحالي
- **التصميم**: ألوان وتصميم مطابق لموقع maabar.io
